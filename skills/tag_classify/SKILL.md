# tag_classify 曲风标签分类技能

作用：对歌曲名称、歌手简介、歌词内容进行分析，自动归纳曲风、使用场景标签。

## 调用场景
原始数据无标签、标签杂乱、多标签合并时使用本技能。

## 执行步骤
1. 接收歌曲名称、歌手、歌词文本
2. 结合音乐常识，输出标准化标签：流行/摇滚/古风/电子/说唱/民谣/纯音乐等
3. 补充场景标签：治愈、热血、怀旧、睡前等
4. 输出结构化标签结果

## 调用示例
```
skill: tag_classify
args: {"title": "七里香", "artist": "周杰伦", "lyric": "窗外的麻雀..."}
```
